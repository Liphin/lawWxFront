PROFESSIONAL ABILITY
专业能力

1、为顾问单位的生产、经营、管理方面重大决策的合法性、可行性、风险预测及对策提供法律依据和法律意见；

2、为顾问单位深化企业改革、完善法人治理结构、加强生产经营管理、提高企业经济效益等有关问题提供法律意见；

3、建立和完善顾问单位的相关规章制度，草拟、修改、审查合同和有关法律文书；

4、受托参与经济合同及有关项目的磋商、谈判，进行法律分析、论证和签约；

5、受托为顾问单位重大项目对外出具法律意见书；

6、代理顾问单位发律师函向债务人或担保人追款；

7、代理顾问单位办理有关股份制改造、股份公司设立以及股票发行上市等法律事务；

8、协助顾问单位处理内部劳动关系；

9、对顾问单位职工进行法制宣传、教育、培训；

10、办理顾问单位其他必要的法律事务。

1、接受委托，为委托人进行企业设立、股权转让、增资减资、招标投标、合并分立、清算注销、资产重组、改制上市、改制、拍卖、清算等全面提供法律服务；

2、接受委托，根据委托人提供的财务资料，对委托人的债权进行分析，对不良资产提出相应的处理方案，清理债权债务；

3、接受委托，为委托人以房地产为载体的资产运作（转让、抵押、拍卖、股权转让等）提供房地产项目全过程法律服务；

4、接受委托，为委托人的相关法律事务提供专项法律服务。

1、接受当事人委托，设立、优化诉讼或仲裁方案，起草各类诉讼、仲裁文书，担任经济、民事、行政案件当事人的委托代理人参加诉讼、仲裁活动；

2、接受委托参与案件的听证、复议、申诉等活动；

3、接受委托代为申请财产保全、申请及协助强制执行各类生效的法律文书（包括裁决书、判决书、处罚决定书、赋于强制执行效力的公证法律文书等）。

4、接受委托，为侦查阶段、审查起诉阶段、提起公诉阶段（一、二审）犯罪嫌疑人（被告人）的辩护人，为嫌疑人（被告人）提供法律帮助、进行辩护。

5、接受委托，担任刑事案件被害人的诉讼代理人或担任刑事附带民事诉讼案件的代理人，代为查阅、搜集资料，代为参与庭审。
